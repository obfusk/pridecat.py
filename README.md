## Requirements

Python >= 3.5 + click.

## Installing

Install the dependencies, as e.g. debian packages or using `pip`:

```bash
$ apt install python3-click       # debian/ubuntu
$ pip install click               # pip
```

Then just put `pridecat` somewhere on your `$PATH` (e.g. `~/bin`).
